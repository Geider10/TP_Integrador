# Clínica Central - FullStack

### Descripción 
Esta aplicación facilita la organización, ayuda a ahorrar tiempo y reducir errores de una clínica.
Permite que los pacientes puedan registrarse, iniciar sesión y gestionar sus turnos de manera sencilla. Al mismo tiempo, los administradores pueden mantener actualizada la información de doctores y especialidades.


### Características
- Autenticación y autorización.
- Encriptación de contraseñas.
- Gestión de turnos

## Funcionalidades por rol
**Paciente**
- Registrarse e iniciar sesión
- Crear, editar y cancelar turnos
- Gestinar datos del perfil

**Admin**
- Gestionar especialidades
- Gestionar doctores

## Tecnologias
- Backend: PHP
- Base de datos: MySQL
- Frontend: HTML, CSS, JavaScript
